010.Go基础 数组

October 16, 2024

在 Go 语言中，数组是一种固定长度、同构（同一类型）的数据集合。数组可以存储同一类型的数据元素，并且数组的长度在声明时就被确定，不能动态改变。Go 语言的数组有明确的类型和长度，因此与一些动态语言中的数组不同，Go 的数组在使用时需要考虑其长度和类型。

1. 数组的定义#

在 Go 语言中，数组通过方括号[]定义，方括号中的数字表示数组的长度，之后跟随数组中元素的类型。数组的长度是数组类型的一部分，因此长度不同的数组是不同的类型。

示例：数组的定义和初始化#

var arr [5]int  // 声明一个长度为5的int类型数组，元素默认值为0
fmt.Println(arr) // 输出 [0 0 0 0 0]

在这个例子中，声明了一个int类型的数组arr，它有 5 个元素，每个元素的初始值为0（因为 Go 中的整数类型默认值是0）。

2. 数组的初始化#

数组在声明的同时可以进行初始化。可以使用字面量的方式为数组赋值，或者只初始化部分元素，剩下的元素会使用该类型的零值进行填充。

示例：数组的初始化#

arr1 := [5]int{1, 2, 3, 4, 5}  // 初始化一个长度为5的数组
fmt.Println(arr1)  // 输出 [1 2 3 4 5]

arr2 := [5]int{1, 2}  // 部分初始化，剩余元素自动赋值为0
fmt.Println(arr2)  // 输出 [1 2 0 0 0]

可以通过这种方式只初始化数组的一部分，未初始化的元素会自动设为该类型的默认值。

3. 使用自动推断长度的数组#

在数组初始化时，可以省略数组的长度，Go 会根据提供的初始值个数自动推断数组的长度。

示例：自动推断数组长度#

arr := [...]int{10, 20, 30, 40}  // Go自动推断数组长度为4
fmt.Println(arr)  // 输出 [10 20 30 40]

在这个例子中，使用...让 Go 自动推断数组的长度。

4. 访问和修改数组元素#

数组的元素通过索引访问。数组的索引从0开始，通过arr[index]的方式可以访问和修改数组的某个元素。

示例：访问和修改数组元素#

arr := [3]string{"apple", "banana", "cherry"}

fmt.Println(arr[0])  // 输出 apple
arr[1] = "orange"    // 修改数组中第二个元素
fmt.Println(arr[1])  // 输出 orange

5. 数组的长度和遍历#

数组的长度是固定的，可以通过内置的len()函数获取数组的长度。遍历数组时，可以使用for循环或range关键字。

示例：获取数组长度和遍历数组#

arr := [...]int{10, 20, 30, 40, 50}

fmt.Println("数组长度为:", len(arr))  // 输出 5

// 使用普通for循环遍历数组
for i := 0; i < len(arr); i++ {
    fmt.Println(arr[i])
}

// 使用range遍历数组
for index, value := range arr {
    fmt.Printf("Index: %d, Value: %d\n", index, value)
}

6. 多维数组#

Go 语言支持多维数组，常见的是二维数组。多维数组的定义与初始化和一维数组类似，只是嵌套了更多的方括号。

示例：二维数组#

var matrix [2][3]int  // 定义一个2行3列的二维数组
matrix = [2][3]int{
    {1, 2, 3},
    {4, 5, 6},
}

fmt.Println(matrix)  // 输出 [[1 2 3] [4 5 6]]

二维数组可以看作是一组数组的数组。使用索引访问时，可以用matrix[row][column]来访问。

7. 数组是值类型#

在 Go 语言中，数组是值类型，这意味着当数组被赋值给另一个数组时，实际上会复制一份新的数组，而不是引用同一个内存位置。数组之间的赋值或传递参数时，会发生值拷贝。

示例：数组是值类型#

arr1 := [3]int{1, 2, 3}
arr2 := arr1  // 复制arr1的值

arr2[0] = 100  // 修改arr2不会影响arr1
fmt.Println(arr1)  // 输出 [1 2 3]
fmt.Println(arr2)  // 输出 [100 2 3]

在这个例子中，arr2是arr1的副本，修改arr2不会影响arr1。

8. 将数组作为函数参数#

因为数组是值类型，数组作为函数参数时，传递的是数组的副本。函数内部的修改不会影响原数组。如果需要在函数中修改数组的内容，可以传递数组的指针。

示例：数组作为函数参数#

func modifyArray(arr [3]int) {
    arr[0] = 100  // 修改的是副本，不影响原数组
}

func modifyArrayPointer(arr *[3]int) {
    arr[0] = 100  // 通过指针修改原数组
}

func main() {
    arr := [3]int{1, 2, 3}

    modifyArray(arr)
    fmt.Println("After modifyArray:", arr)  // 输出 [1 2 3]

    modifyArrayPointer(&arr)
    fmt.Println("After modifyArrayPointer:", arr)  // 输出 [100 2 3]
}

9. 数组与切片#

Go 的数组虽然功能强大，但由于长度固定，在实际编程中使用较少。Go 更常用的是切片（slice），切片是对数组的抽象，可以动态改变大小。虽然切片底层依赖数组，但切片的大小是可变的，更加灵活。

示例：数组与切片的关系#

arr := [5]int{1, 2, 3, 4, 5}
slice := arr[1:4]  // 创建一个切片，引用数组的部分元素
fmt.Println(slice)  // 输出 [2 3 4]

在这个例子中，slice是从数组arr中创建的切片，它引用了数组的第 2 到第 4 个元素。

总结#

  • 数组的定义：Go 中的数组是固定长度的，声明时长度不可更改。
  • 数组初始化：可以使用字面量直接初始化数组，未赋值的元素会使用默认零值。
  • 访问与修改：通过索引访问数组的元素，数组的长度可以使用len()函数获取。
  • 数组是值类型：数组赋值或传递时会复制整个数组，而不是引用同一个内存地址。
  • 多维数组：Go 支持多维数组，常见的有二维数组。
  • 数组与切片：Go 中的切片是对数组的抽象，大小可变，更加灵活，因此切片比数组使用更为广泛。
