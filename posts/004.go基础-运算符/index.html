004.GO基础 运算符

October 16, 2024

Go 语言的运算符主要分为以下几大类：算术运算符、关系运算符、逻辑运算符、位运算符、赋值运算符和其他运算符。每类运算符都用于特定的操作，帮助程序完成各种计算和逻辑判断。下面是详细介绍：

1. 算术运算符（Arithmetic Operators）#

算术运算符用于执行基本的数学运算。

运算符 描述 示例 结果
+ 加法 5 + 3 8
- 减法 5 - 3 2
* 乘法 5 * 3 15
/ 除法 5 / 2 2（整除）
% 取余 5 % 2 1
++ 自增 a++ 等同于 a = a + 1
-- 自减 a-- 等同于 a = a - 1

注意：

  • ++ 和 -- 是 后缀运算符，即先返回当前值，再自增或自减。
  • 整数除法会丢弃小数部分，例如 5 / 2 的结果为 2，而不是 2.5。

2. 关系运算符（Relational Operators）#

关系运算符用于比较两个值，返回 true 或 false。

运算符 描述 示例 结果
== 等于 5 == 5 true
!= 不等于 5 != 3 true
> 大于 5 > 3 true
< 小于 5 < 3 false
>= 大于等于 5 >= 5 true
<= 小于等于 3 <= 5 true

3. 逻辑运算符（Logical Operators）#

逻辑运算符用于布尔值之间的逻辑判断。

运算符 描述 示例 结果
&& 逻辑与（AND） true && false false
|| 逻辑或（OR） true || false true
! 逻辑非（NOT） !true false

注意：

  • && 和 || 是短路运算符，当第一个表达式已经确定整个表达式的值时，第二个表达式将不会被执行。

4. 位运算符（Bitwise Operators）#

位运算符用于对整数类型的位进行操作。

运算符 描述 示例 结果
& 按位与 5 & 3 1（二进制：101 & 011 = 001）
| | 按位或 5 | 3 7（二进制：101 | 011 = 111）
^ 按位异或 5 ^ 3 6（二进制：101 ^ 011 = 110）
&^ 按位清空（AND NOT） 5 &^ 3 4（二进制：101 &^ 011 = 100）
<< 左移 5 << 1 10（二进制：101 « 1 = 1010）
>> 右移 5 >> 1 2（二进制：101 » 1 = 10）

解释：

  • 按位与（&）：对应位都为 1 时结果才为 1。
  • 按位或（|）：对应位有一个为 1 时结果为 1。
  • 按位异或（^）：对应位相异时结果为 1。
  • 按位清空（&^）：将第一个操作数中对应第二个操作数为 1 的位清零。
  • 左移（<<）：将二进制位向左移动指定的位数，右侧补 0。
  • 右移（>>）：将二进制位向右移动指定的位数，左侧补 0（对无符号整数）或补符号位（对有符号整数）。

5. 赋值运算符（Assignment Operators）#

赋值运算符用于给变量赋值，可以与其他运算符组合使用。

运算符 描述 示例 等价表达式
= 赋值 x = 5 x = 5
+= 加后赋值 x += 3 x = x + 3
-= 减后赋值 x -= 2 x = x - 2
*= 乘后赋值 x *= 2 x = x * 2
/= 除后赋值 x /= 2 x = x / 2
%= 取余后赋值 x %= 2 x = x % 2
<<= 左移后赋值 x <<= 2 x = x << 2
>>= 右移后赋值 x >>= 2 x = x >> 2
&= 按位与后赋值 x &= 2 x = x & 2
= 按位或后赋值 x = 2 x = x 2
^= 按位异或后赋值 x ^= 2 x = x ^ 2
&^= 按位清空后赋值 x &^= 2 x = x &^ 2

6. 其他运算符#

6.1. 地址运算符（Address Operator）#

  • &：取变量的内存地址。 
    var x int = 42
var ptr *int = &x  // ptr 存储 x 的地址

6.2. 指针解引用运算符（Pointer Dereference Operator）#

  • *：访问指针指向的值。 
    var x int = 42
var ptr *int = &x
fmt.Println(*ptr)  // 输出 42

6.3. 长度和容量运算符#

  • len：返回数组、切片、字符串或映射的长度。 
    str := "hello"
fmt.Println(len(str))  // 输出 5
  • cap：返回数组或切片的容量。 
    slice := make([]int, 5, 10)
fmt.Println(cap(slice))  // 输出 10

总结#

Go 语言提供了丰富的运算符，涵盖了基本的算术操作、逻辑操作、位操作和赋值操作。通过这些运算符，可以高效地执行各种计算和操作，编写简洁明了的代码。
