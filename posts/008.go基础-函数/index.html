008.Go基础 函数

October 16, 2024

在 Go 语言中，函数是程序的基本构建块之一。函数定义了可复用的代码块，通过传递参数和返回值来实现逻辑封装。Go 语言的函数具有灵活和强大的特性，如支持多返回值、匿名函数、闭包以及延迟执行（defer）等。接下来，我将介绍 Go 语言中的函数特性及其使用方式。

1. 基本函数定义#

Go 语言中，函数的定义使用关键字func，并且参数和返回值的类型必须显式声明。

func 函数名(参数列表) 返回值类型 {
    // 函数体
}

示例：基本函数#

func add(x int, y int) int {
    return x + y
}

func main() {
    result := add(3, 4)
    fmt.Println(result)  // 输出 7
}

在这个示例中，add函数接受两个int类型的参数，并返回它们的和。

2. 多返回值函数#

Go 语言允许函数返回多个值。这在需要返回结果和错误状态的情况下特别有用。

示例：多返回值#

func swap(x, y string) (string, string) {
    return y, x
}

func main() {
    a, b := swap("hello", "world")
    fmt.Println(a, b)  // 输出 world hello
}

在这个示例中，swap函数返回两个字符串，通过多重赋值操作接收返回值。

3. 命名返回值#

函数的返回值可以被命名，命名返回值相当于在函数开始时声明的变量，可以在函数体内直接使用。命名返回值可以使代码更简洁，并且在返回值上也能有更好的可读性。

示例：命名返回值#

func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // 没有参数，返回命名的返回值
}

func main() {
    fmt.Println(split(17))  // 输出 7 10
}

在这个例子中，x和y是命名返回值，函数体内对它们赋值后直接使用return返回。

4. 变长参数#

Go 语言支持变长参数（可变参数），也就是参数个数不固定的情况。使用...表示变长参数，变长参数实际被视作切片。

示例：变长参数#

func sum(nums ...int) int {
    total := 0
    for _, num := range nums {
        total += num
    }
    return total
}

func main() {
    fmt.Println(sum(1, 2, 3))        // 输出 6
    fmt.Println(sum(4, 5, 6, 7, 8))  // 输出 30
}

在这个例子中，函数sum可以接受任意数量的整数参数。

5. 匿名函数和闭包#

匿名函数是在没有名称的情况下定义的函数。匿名函数可以赋值给变量，作为参数传递，或者作为闭包捕获外部变量。

示例：匿名函数#

func main() {
    f := func(x, y int) int {
        return x + y
    }
    fmt.Println(f(3, 4))  // 输出 7
}

示例：闭包#

闭包是一个函数值，它能够引用其外部作用域中的变量，即使这个变量在闭包执行时已经超出了其原本的作用范围。

func adder() func(int) int {
    sum := 0
    return func(x int) int {
        sum += x
        return sum
    }
}

func main() {
    pos := adder()
    fmt.Println(pos(1))  // 输出 1
    fmt.Println(pos(2))  // 输出 3
    fmt.Println(pos(3))  // 输出 6
}

在这个例子中，返回的匿名函数捕获了adder函数中的sum变量，即使adder函数已经返回，sum依然在每次调用时被更新。

6. defer延迟执行#

defer语句用于将函数调用延迟到当前函数执行完毕后再执行。常用于资源清理操作，例如关闭文件、释放锁等。

示例：defer的基本用法#

func main() {
    defer fmt.Println("world")
    fmt.Println("hello")
}

在这个例子中，defer语句会在main函数即将返回时执行，因此输出顺序是hello之后才是world。

示例：多个defer语句#

多个defer语句会按后进先出的顺序执行：

func main() {
    defer fmt.Println("1")
    defer fmt.Println("2")
    defer fmt.Println("3")
}

输出顺序为：

3
2
1

7. 方法（Method）#

Go 语言的函数还可以与类型关联，这种函数称为方法。通过定义接收者（receiver），函数可以作用于某种类型的变量。

示例：方法定义#

type Person struct {
    Name string
}

func (p Person) SayHello() {
    fmt.Println("Hello, my name is", p.Name)
}

func main() {
    p := Person{Name: "Alice"}
    p.SayHello()  // 输出 Hello, my name is Alice
}

在这个例子中，SayHello是Person类型的一个方法，p Person是方法的接收者。

总结#

  • 基本函数：通过func关键字定义，支持参数和返回值。
  • 多返回值：Go 支持函数返回多个值。
  • 变长参数：使用...可以传递不定数量的参数。
  • 匿名函数和闭包：匿名函数没有名称，可以形成闭包捕获外部变量。
  • defer延迟执行：常用于延迟清理操作。
  • 方法：Go 语言的函数可以与类型关联，通过接收者定义方法。

Go 语言的函数功能灵活且强大，支持各种复杂的编程需求。
