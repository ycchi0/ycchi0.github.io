005.Go基础 分支语句

October 16, 2024

Go 语言中的分支结构用于根据条件执行不同的代码路径。常见的分支结构有 if 语句、switch 语句和特殊的 select 语句。每种分支结构都有其适用场景，下面我将详细介绍它们。

1. if 语句#

if 语句是最常用的条件分支结构，依据一个布尔表达式的结果来决定是否执行某段代码。Go 语言的 if 语句语法简单，且支持可选的初始化语句。

语法：#

if condition {
    // 当 condition 为 true 时执行
}

示例：基本 if 语句#

package main

import "fmt"

func main() {
    x := 10
    if x > 5 {
        fmt.Println("x 大于 5")
    }
}

示例：带初始化语句的 if#

Go 语言的 if 语句支持在条件判断前执行一个短语句，这个特性通常用于声明和初始化局部变量。

if x := 10; x > 5 {
    fmt.Println("x 大于 5")
}
  • x 只在 if 语句的作用域内可见。

if-else 语句#

if condition {
    // 当 condition 为 true 时执行
} else {
    // 当 condition 为 false 时执行
}

示例：if-else 语句#

x := 3
if x > 5 {
    fmt.Println("x 大于 5")
} else {
    fmt.Println("x 小于或等于 5")
}

if-else if-else 语句#

多个条件可以通过 else if 组合，来实现多分支的逻辑判断。

if condition1 {
    // 当 condition1 为 true 时执行
} else if condition2 {
    // 当 condition2 为 true 时执行
} else {
    // 当以上条件均不满足时执行
}

示例：if-else if-else 语句#

x := 8
if x < 5 {
    fmt.Println("x 小于 5")
} else if x < 10 {
    fmt.Println("x 小于 10")
} else {
    fmt.Println("x 大于或等于 10")
}

2. switch 语句#

switch 语句用于替代多个 if-else if 判断，常用于对单个变量或表达式的多个值进行判断。与其他语言不同，Go 的 switch 语句无需显式使用 break 关键字来终止一个分支。

语法：#

switch expression {
case value1:
    // 当 expression 等于 value1 时执行
case value2:
    // 当 expression 等于 value2 时执行
default:
    // 当 expression 不匹配任何 case 时执行
}

示例：基本 switch 语句#

package main

import "fmt"

func main() {
    x := 2
    switch x {
    case 1:
        fmt.Println("x 是 1")
    case 2:
        fmt.Println("x 是 2")
    case 3:
        fmt.Println("x 是 3")
    default:
        fmt.Println("x 不是 1、2 或 3")
    }
}

无表达式的 switch#

Go 的 switch 还可以不带表达式使用，这时它会将每个 case 视为布尔表达式，类似多个 if-else。

switch {
case condition1:
    // condition1 为 true 时执行
case condition2:
    // condition2 为 true 时执行
default:
    // 所有条件为 false 时执行
}

示例：无表达式的 switch#

x := 10
switch {
case x < 5:
    fmt.Println("x 小于 5")
case x < 10:
    fmt.Println("x 小于 10")
default:
    fmt.Println("x 大于等于 10")
}

多个值的 case#

在一个 case 中可以同时匹配多个值。

switch x {
case 1, 3, 5:
    fmt.Println("x 是奇数")
case 2, 4, 6:
    fmt.Println("x 是偶数")
}

fallthrough 关键字#

在 Go 的 switch 语句中，每个 case 执行完后会自动退出 switch，不需要 break。但如果希望在匹配成功后继续执行下一个 case，可以使用 fallthrough 关键字。

示例：fallthrough 使用#

x := 1
switch x {
case 1:
    fmt.Println("x 是 1")
    fallthrough
case 2:
    fmt.Println("fallthrough 到 2")
}

输出：

x 是 1
fallthrough 到 2

3. select 语句#

select 语句是 Go 中用于处理多个通道操作的特殊分支结构。它会在多个通道操作中随机选择一个可以执行的分支。select 语句与 switch 类似，但用于通道的并发操作。

语法：#

select {
case <-chan1:
    // 当 chan1 可读时执行
case chan2 <- value:
    // 当 chan2 可写时执行
default:
    // 当没有通道可用时执行
}

示例：select 语句#

package main

import (
    "fmt"
    "time"
)

func main() {
    chan1 := make(chan string)
    chan2 := make(chan string)

    go func() {
        time.Sleep(1 * time.Second)
        chan1 <- "来自通道 1 的消息"
    }()

    go func() {
        time.Sleep(2 * time.Second)
        chan2 <- "来自通道 2 的消息"
    }()

    select {
    case msg1 := <-chan1:
        fmt.Println(msg1)
    case msg2 := <-chan2:
        fmt.Println(msg2)
    }
}

在这个例子中，select 会等待任意一个通道变得可用，然后执行对应的 case 语句。

总结#

  • if 语句：用于根据条件判断来执行不同的代码块，可带初始化语句，支持 if-else 和 if-else if-else 的多分支逻辑。
  • switch 语句：简化了多个条件判断的写法，Go 的 switch 语句无需 break，且支持多值匹配、无表达式的情况以及 fallthrough 关键字。
  • select 语句：是专门为处理多个通道操作设计的分支结构，用于在并发程序中选择处理多个通道的操作。
