002.Go基础 基础数据类型

October 16, 2024

Go 语言是一种静态强类型语言，提供了多种基础数据类型，主要分为 布尔型、数字型、字符串型 和 特殊类型（byte 和 rune），此外还有一些常用的复合类型。以下是详细介绍：

1. 布尔型（Boolean）#

  • 类型： bool
  • 取值： true 或 false
  • 示例： 
    var isTrue bool = true
var isFalse bool = false

2. 数字类型（Numeric Types）#

数字类型分为 整数类型、浮点数类型 和 复数类型。

2.1. 整数类型（Integer Types）#

有符号整数：#

  • 类型： int8, int16, int32, int64
  • 范围：
    • int8: -128 到 127
    • int16: -32,768 到 32,767
    • int32: -2,147,483,648 到 2,147,483,647
    • int64: -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807
  • 示例： 
    var a int8 = -100
var b int32 = 10000

无符号整数：#

  • 类型： uint8, uint16, uint32, uint64
  • 范围：
    • uint8: 0 到 255
    • uint16: 0 到 65,535
    • uint32: 0 到 4,294,967,295
    • uint64: 0 到 18,446,744,073,709,551,615
  • 示例： 
    var c uint8 = 200
var d uint32 = 40000

机器字长整数：#

  • 类型： int, uint
  • 范围： 取决于系统架构（32 位或 64 位）。
  • 示例： 
    var e int = -50
var f uint = 50

2.2. 浮点数类型（Floating-Point Types）#

  • 类型： float32, float64
  • 范围：
    • float32: 精度约为 7 位十进制数
    • float64: 精度约为 15 位十进制数
  • 示例： 
    var x float32 = 3.14
var y float64 = 3.14159265359

2.3. 复数类型（Complex Types）#

  • 类型： complex64, complex128
  • 示例： 
    var z complex64 = 1 + 2i
var w complex128 = 2.5 + 3.7i

3. 字符串类型（String）#

  • 类型： string
  • 描述： Go 中的字符串是 UTF-8 编码的字节序列，字符串是不可变的。
  • 示例： 
    var str string = "Hello, 世界"

4. 特殊类型#

4.1. byte 类型#

  • 类型： byte 是 uint8 的别名，表示一个 ASCII 字符或原始字节。
  • 示例： 
    var b byte = 'A'  // 'A' 的 ASCII 码值是 65

4.2. rune 类型#

  • 类型： rune 是 int32 的别名，用于表示 Unicode 字符。

  • 示例：

    var r rune = '中'  // '中' 的 Unicode 编码值是 20013

  • 字符串中 byte 和 rune 的应用：

    • 逐字节遍历： 使用 byte
    • 逐字符遍历： 使用 rune

    示例：

    // 逐字节遍历
for i := 0; i < len(str); i++ {
    fmt.Printf("%c ", str[i])
}

// 逐字符遍历
for _, r := range str {
    fmt.Printf("%c ", r)
}

5. 复合类型（Composite Types）#

5.1. 数组（Array）#

  • 固定长度的同类型元素集合。 
    var arr [5]int = [5]int{1, 2, 3, 4, 5}

5.2. 切片（Slice）#

  • 动态长度的数组。 
    var slice []int = []int{1, 2, 3, 4, 5}

5.3. 映射（Map）#

  • 键值对数据结构，类似哈希表。 
    var m map[string]int = map[string]int{"apple": 2, "banana": 3}

5.4. 结构体（Struct）#

  • 自定义数据结构，包含多个字段。 
    type Person struct {
    Name string
    Age  int
}

总结#

Go 语言提供了丰富的基础数据类型，从基本的布尔、整数、浮点数和字符串到处理字符的 byte 和 rune，它们是开发者编写高效、简洁代码的基础。
